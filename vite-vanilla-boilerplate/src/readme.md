# Image Feed App

A web application that fetches and displays images from the Image Feed API using only HTML, CSS, and JavaScript. All photos are loaded dynamically, and the interface includes interactive elements to enhance the user experience.

## Overview

The app retrieves images from the API and displays them in a layout inspired by modern image platforms. No frameworks or external libraries are used, only pure JavaScript for fetching data and controlling interactivity.

## Features

- Fetch and display photos from the Image Feed API  
- Render all content dynamically with JavaScript  
- Include interactive feature
- Styled with CSS and structured with semantic HTML  

## Technologies Used

HTML, CSS, JavaScript, Git, GitHub
